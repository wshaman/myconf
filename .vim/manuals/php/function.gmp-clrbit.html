gmp_clrbit

(PHP 4 >= 4.0.4, PHP 5)

gmp_clrbit — Clear bit

Описание

void gmp_clrbit ( resource $a , int $index )

Clears (sets to 0) bit index in a. The index starts at 0.

Список параметров

a

Может быть как числом GMP, так строкой, учитывая то, что ее значение правильно конвертируется в число.

index

Может быть как числом GMP, так строкой, учитывая то, что ее значение правильно конвертируется в число.

Возвращаемые значения

Число GMP.

Примеры

Пример #1 gmp_clrbit() example

<?php
$a = gmp_init("0xff");
gmp_clrbit($a, 0); // index starts at 0, least significant bit
echo gmp_strval($a) . "\n";
?>

Результат выполнения данного примера:

254

Примечания

Замечание:

Unlike most of the other GMP functions, gmp_clrbit() must be called with a GMP resource that already exists (using gmp_init() for example). One will not be automatically created.

Смотрите также

  • gmp_setbit() - Set bit
  • gmp_testbit() - Tests if a bit is set

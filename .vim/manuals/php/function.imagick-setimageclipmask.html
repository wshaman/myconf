Imagick::setImageClipMask

(No version information available, might only be in SVN)

Imagick::setImageClipMask — Sets image clip mask

Описание

bool Imagick::setImageClipMask ( Imagick $clip_mask )

Sets image clip mask from another Imagick object. Этот метод доступен, если Imagick был скомпилирован с версией ImageMagick 6.3.6 или старше.

Список параметров

clip_mask

The Imagick object containing the clip mask

Возвращаемые значения

В случае успешной работы возвращает TRUE.

Ошибки

Бросает ImagickException при ошибке.

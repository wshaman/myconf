http_post_fields

(PECL pecl_http >= 0.10.0)

http_post_fields — Perform POST request with data to be encoded

Описание

string http_post_fields ( string $url , array $data [, array $files [, array $options [, array &$info ]]] )

Performs an HTTP POST request on the supplied url.

Смотрите полный список опции запроса.

Список параметров

url

URL

data

Associative array of POST values

files

Array of files to post

options

опции запроса

info

Информация о запросе/ответе

Возвращаемые значения

Возвращает строку с HTTP-ответом(ами) при успешном завершении работы, или FALSE при ошибке.

Примеры

Пример #1 A http_post_fields() example

<?php
$fields = array(
    'name' => 'mike',
    'pass' => 'se_ret'
);
$files = array(
    array(
        'name' => 'uimg',
        'type' => 'image/jpeg',
        'file' => './profile.jpg',
    )
);

$response = http_post_fields("http://www.example.com/", $fields, $files);
?>

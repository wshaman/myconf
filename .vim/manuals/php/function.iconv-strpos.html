iconv_strpos

(PHP 5)

iconv_strpos — Возвращает позицию первого вхождения подстроки

Описание

int iconv_strpos ( string $haystack , string $needle [, int $offset [, string $charset ]] )

Возвращает позицию первого вхождения подстроки needle в строку haystack.

Необязательный аргумент offset позволяет указать, с какого по счёту символа строки начинать поиск.

Если подстрока needle не найдена, iconv_strpos() возвращает FALSE.

Внимание

Эта функция может возвращать как boolean FALSE, так и не-boolean значение, которое приводится к FALSE, например, 0 или "". За более подробной информации обратитесь к разделу Булев тип. Используйте оператор === для проверки значения, возвращаемого этой функцией.

Если haystack или needle не является строкой, он приводится к целому и трактуется как код символа.

В отличие от strpos(), iconv_strpos() возвращает количество символов до подстроки, в не позицию вхождения в байтах. Символы считаются с учётом их кодировки charset.

Если параметр charset опущен, предполагается, что кодировка строки string равна iconv.internal_charset.

См. также strpos(), iconv_strrpos() и mb_strpos().

register_tick_function

(PHP 4 >= 4.0.3, PHP 5)

register_tick_function — Регистрирует функцию для выполнения при каждом тике

Описание

bool register_tick_function ( callback $function [, mixed $arg [, mixed $... ]] )

Регистрирует функцию function, которая должна вызываться при каждом тике.

Список параметров

function

Строка с именем функции, или массив, состоящий из объекта и метода.

arg

...

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования register_tick_function()

<?php
// using a function as the callback
register_tick_function('my_function', true);

// using an object->method
$object = new my_class();
register_tick_function(array(&$object, 'my_method'), true);
?>

Список изменений

Версия Описание
5.3.0 Тики теперь поддерживаются и на потоковых модулях веб-серверов.

Примечания

Внимание

Не используйте register_tick_function() на потоковых модулях веб-серверов с PHP 5.2 и ниже.

XMLReader::readOuterXML

(PHP 5 >= 5.2.0)

XMLReader::readOuterXML — Retrieve XML from current node, including it self

Описание

string XMLReader::readOuterXML ( void )

Reads the contents of the current node, including the node itself.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Returns the contents of current node, including itself, as a string. Empty string on failure.

Примечания

Предостережение

Эта функция доступна только если PHP скомпилирован с поддержкой libxml 20620 или старше.

Смотрите также

  • XMLReader::readString() - Reads the contents of the current node as an string
  • XMLReader::readInnerXML() - Retrieve XML from current node
  • XMLReader::expand() - Returns a copy of the current node as a DOM object

gzpassthru

(PHP 4, PHP 5)

gzpassthru — Выводит все оставшиеся данные из указателя gz-файла.

Описание

int gzpassthru ( resource $zp )

Читает до конца файла (EOF) данные из указателя gz-файла и записывает (несжатый) результат в стандартный вывод.

Замечание:

Используйте gzrewind(), чтобы переместить позицию в начало файла, если вы уже записали данные в него.

Подсказка

Если вам нужно просто вывести содержимое файла без перемещения позиции или внесения изменений, используйте функцию readgzfile(), которая не требует вызова функции gzopen().

Список параметров

zp

Указатель на gz-файл, возвращённый, после его успешного открытия, функцией gzopen().

Возвращаемые значения

Количество символов (после распаковки), прочтённых из gz, или FALSE в случае ошибки.

Примеры

Пример #1 Пример использования gzpassthru()

<?php
$fp = gzopen('file.gz', 'r');
gzpassthru($fp);
gzclose($fp);
?>

sqlite_column

(PHP 5, PECL sqlite >= 1.0.0)

sqlite_column — Выбирает одно поле из текущей записи результата запроса

Описание

mixed sqlite_column ( resource $result , mixed $index_or_name [, bool $decode_binary ] )

Выбирает значение одного поля из текущей записи результата запроса result. Аргумент index_or_name задает либо имя поля (если передана строка), либо порядковый номер поля (если передано целое). Описание аргумента decode_binary см. в описании функции sqlite_fetch_array().

Эта функция используется с запросами, возвращающими большие наборы данных с большим количеством полей, либо если поля содержат большие объемы данных.

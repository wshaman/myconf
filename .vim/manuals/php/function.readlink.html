readlink

(PHP 4, PHP 5)

readlink — Возвращает файл, на который указывает символическая ссылка

Описание

string readlink ( string $path )

readlink() делает то же самое, что и функция C readlink - возвращает содержимое пути символической ссылки или FALSE в случае ошибки.

Пример #1 Пример использования функции readlink()

<?php

// output e.g. /boot/vmlinux-2.4.20-xfs
echo readlink('/vmlinuz');

?>

Замечание: Для Windows-платформ эта функция не реализована.

См. также описание функций is_link(), symlink() и linkinfo().

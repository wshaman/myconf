gzdeflate

(PHP 4 >= 4.0.4, PHP 5)

gzdeflate — Сжимает строку

Описание

string gzdeflate ( string $data [, int $level ] )

Сжимает строку используя формат данных DEFLATE.

Для подробностей по алгоритму сжатия DEFLATE, см. "» DEFLATE Compressed Data Format Specification 1.3" (RFC 1951).

Список параметров

data

Данные для сжатия.

level

Уровень сжатия. 0 - без сжатия, 9 - максимум. Если не указан, будет использоваться уровень сжатия по умолчанию библиотеки zlib.

Возвращаемые значения

Сжатая строка или FALSE в случае ошибки.

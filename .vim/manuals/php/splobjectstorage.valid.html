SplObjectStorage::valid

(PHP 5 >= 5.1.0)

SplObjectStorage::valid — Returns if the current iterator entry is valid

Описание

public bool SplObjectStorage::valid ( void )

Returns if the current iterator entry is valid.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Returns TRUE if the iterator entry is valid, FALSE otherwise.

Примеры

Пример #1 SplObjectStorage::valid() example

<?php
$s = new SplObjectStorage();

$o1 = new StdClass;
$o2 = new StdClass;

$s->attach($o1, "d1");
$s->attach($o2, "d2");

$s->rewind();
while($s->valid()) {
    echo $s->key()."\n";
    $s->next();
}
?>

dngettext

(PHP 4 >= 4.2.0, PHP 5)

dngettext — Версия dgettext для множественного числа

Описание

string dngettext ( string $domain , string $msgid1 , string $msgid2 , int $n )

Запрашивает перевод с чиловым параметром из другого каталога сообщений.

См. также ngettext().

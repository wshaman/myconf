sqlite_busy_timeout

(PHP 5, PECL sqlite >= 1.0.0)

sqlite_busy_timeout — Устанавливает время ожидания

Описание

void sqlite_busy_timeout ( resource $dbhandle , int $milliseconds )

Устанавливает максимальное время, в течение которого sqlite ожидает готовности базы данных, заданной аргументом dbhandle, в milliseconds миллисекунд. Если milliseconds равен 0, то в случае, когда файл базы данных заблокирован другим процессом, большинство функций sqlite будут незамедлительно возвращать значение SQLITE_BUSY.

При открытии базы данных,PHP устанавливает время ожидания в 60 секунд.

Замечание:

Обратите внимание, что время ожидания задается не в секундах, а в миллисекундах.

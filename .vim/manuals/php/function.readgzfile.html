readgzfile

(PHP 4, PHP 5)

readgzfile — Выводит содержимое gz-файла

Описание

int readgzfile ( string $filename [, int $use_include_path ] )

Читает файл, распаковывает его и записывает в стандартный вывод.

Может также использоваться для чтения файла, не имеющего формат gzip; в этом случае readgzfile() непосредственно читает из файла без дополнительной обработки.

Список параметров

filename

Имя файла.

use_include_path

Вы можете использовать необязательный второй параметр и установить его в 1, если вы хотите, чтобы файл также искался в директориях include_path.

Возвращаемые значения

Возвращает количество байтов в файле (после распаковки). Если возникает ошибка, возвращается FALSE и, если функция не была вызвана как @readgzfile, выводится сообщение об ошибке.

Смотрите также

  • gzpassthru() - Выводит все оставшиеся данные из указателя gz-файла.
  • gzfile() - Считывает весь gz-файл в массив
  • gzopen() - Открывает gz-файл

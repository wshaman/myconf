PECL для пользователей Windows

Как и в случае DLL расширения PHP, установка проста и заключается в копировании файла DLL расширения PECL в директорию extension_dir и подключением затем его через php.ini. Например, добавьте следующую строку в ваш php.ini:


extension=php_extname.dll

После выполнения этих действий, перезапустите web-сервис.

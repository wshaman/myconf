lob->tell

(No version information available, might only be in SVN)

lob->tell — Возвращает текущую позицию внутреннего указателя объекта

Описание

int lob->tell ( void )

Возвращает текущую позицию внутреннего указателя объекта LOB или FALSE в случае ошибки.

См. также oci_lob_size() и oci_lob_eof().

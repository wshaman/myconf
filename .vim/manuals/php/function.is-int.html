is_int

(PHP 4, PHP 5)

is_int — Проверяет, является ли переменная целого типа

Описание

bool is_int ( mixed $var )

Проверяет, является ли данная переменная целого типа.

Замечание:

Чтобы проверить, что переменная является числом или числовой строкой (как поле ввода в форме, которое всегда является строкой), Вы должны использовать is_numeric().

Список параметров

var

Проверяемая переменная.

Возвращаемые значения

Возвращает TRUE, если var является целым числом, FALSE в противном случае.

Примеры

Пример #1 Пример использования is_int()

<?php
if (is_int(23)) {
 echo "целое число\n";
} else {
 echo "не целое число\n";
}
var_dump(is_int(23));
var_dump(is_int("23"));
var_dump(is_int(23.5));
var_dump(is_int(true));
?>

Результат выполнения данного примера:

целое число
bool(true)
bool(false)
bool(false)
bool(false)

Смотрите также

  • is_bool() - Проверяет, является ли переменная булевой
  • is_float() - Проверяет, является ли переменная числом с плавающей точкой
  • is_numeric() - Проверяет, является ли переменная числом или числовой строкой
  • is_string() - Проверяет, является ли переменная строкой
  • is_array() - Определяет, является ли переменная массивом
  • is_object() - Проверяет, является ли переменная объектом

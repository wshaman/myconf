checkdate

(PHP 4, PHP 5)

checkdate — Проверяет правильность даты по грегорианскому календарю

Описание

bool checkdate ( int $month , int $day , int $year )

Проверяет правильность даты, указанной переданными аргументами. Дата считается верной, если верны все переданные параметры.

Список параметров

month

Месяц, может принимать значения от 1 до 12 включительно.

day

Допустимый день для указанного месяца month. Параметр year также учитывает високосные года.

year

Год, может принимать значения от 1 до 32767 включительно.

Возвращаемые значения

Возвращает TRUE если указанной дата является правильной, иначе возвращает FALSE.

Примеры

Пример #1 Пример использования функции checkdate()

<?php
var_dump(checkdate(12, 31, 2000));
var_dump(checkdate(2, 29, 2001));
?>

Результат выполнения данного примера:

bool(true)
bool(false)

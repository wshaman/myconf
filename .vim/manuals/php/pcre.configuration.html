Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Опции конфигурации PCRE
Имя Значение по умолчанию Изменяемость Changelog
pcre.backtrack_limit "100000" PHP_INI_ALL Доступна с PHP 5.2.0.
pcre.recursion_limit "100000" PHP_INI_ALL Доступна с PHP 5.2.0.
Для подробного описания констант PHP_INI_*, обратитесь к разделу Where a configuration setting may be set.

Краткое разъяснение конфигурационных директив.

pcre.backtrack_limit integer

Лимит обратных ссылок PCRE.

pcre.recursion_limit integer

Лимит на рекурсию. Не забывайте о том, что если вы установите достаточно высокое значение, то PCRE может превысить размер стэка (установленный операционной системой) и в конце концов вызовет обрушение PHP.

Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Параметры конфигурации Variables
Имя Значение по умолчанию Область изменения Changelog
unserialize_callback_func NULL PHP_INI_ALL Доступна с PHP 4.2.0.
Для подробного описания констант PHP_INI_*, обратитесь к разделу Where a configuration setting may be set.

Краткое разъяснение конфигурационных директив.

unserialize_callback_func string

Функция обратного вызова для unserialize() будет вызвана (с неопределенным именем класса в качестве параметра), если в процессе рассериализации будет обнаружен неопределенный класс, который должен быть проинициализирован. Если указанная функция не определена или если она не подключает/реализовывает отсутствующий класс. Поэтому устанавливайте это значение только в том случае, если Вы действительно хотите реализовать подобную функцию обратного вызова.

См. также unserialize().

ReflectionFunction::export

(PHP 5)

ReflectionFunction::export — Exports function

Описание

public static string ReflectionFunction::export ( string $name [, string $return ] )

Exports a Reflected function.

Список параметров

name

Экспортируемый объект Reflection.

return

Установка в TRUE вернет экспортируемое значение, в отличие от поведения, где этот параметр опущен. Установка в FALSE (по умолчанию) сделает противоположное.

Возвращаемые значения

Если параметр return установлен в TRUE, тогда экспортируемый объект будет возвращен как string, иначе будет возвращен NULL.

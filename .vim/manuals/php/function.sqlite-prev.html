sqlite_prev

(PHP 5)

sqlite_prev — Переходит на предыдущую запись

Описание

bool sqlite_prev ( resource $result )

sqlite_prev() перемещает внутренний указатель результата запроса result на предыдущую запись. Возвращает FALSE если указатель находится на первой записи в момент вызова функции, иначе TRUE.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

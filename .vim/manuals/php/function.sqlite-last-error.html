sqlite_last_error

(PHP 5, PECL sqlite >= 1.0.0)

sqlite_last_error — Возвращает код ошибки последней операции с SQLite

Описание

int sqlite_last_error ( resource $dbhandle )

Возвращает код ошибки последней операции с базой данных с идентификатором dbhandle. Описание ошибки можно получить с помощью функции sqlite_error_string().

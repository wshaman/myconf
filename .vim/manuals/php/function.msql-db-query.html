msql_db_query

(PHP 4, PHP 5)

msql_db_query — Send mSQL query

Описание

resource msql_db_query ( string $database , string $query [, resource $link_identifier ] )

msql_db_query() selects a database and executes a query on it.

Список параметров

database

The name of the mSQL database.

query

The SQL query.

link_identifier

Соединение mSQL. Если не указан, используется последнее соединение, открытое msql_connect(). Если такое соединение не было найдено, функция попытается установить новое, как если бы была вызвана функция msql_connect(), и использовать его.

Возвращаемые значения

Returns a positive mSQL query identifier to the query result, or FALSE on error.

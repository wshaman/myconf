msql_field_table

(PHP 4, PHP 5)

msql_field_table — Get table name for field

Описание

int msql_field_table ( resource $result , int $field_offset )

Returns the name of the table that the specified field is in.

Список параметров

result

Обрабатываемый результат запроса, полученный с помощью функции msql_query().

field_offset

Числовое смещение поля. field_offset начинается с 1.

Возвращаемые значения

The name of the table on success или FALSE в случае возникновения ошибки.

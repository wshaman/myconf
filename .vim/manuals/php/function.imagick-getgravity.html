Imagick::getGravity — Gets the gravity

Описание

int Imagick::getGravity ( void )

Gets the global gravity property for the Imagick object. Этот метод доступен, если Imagick был скомпилирован с версией ImageMagick 6.4.0 или старше.

Возвращаемые значения

Returns the gravity property. Refer to the list of gravity constants.

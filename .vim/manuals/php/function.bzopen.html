bzopen

(PHP 4 >= 4.0.4, PHP 5)

bzopen — Открывает файл, сжатый с использованием bzip2

Описание

resource bzopen ( string $filename , string $mode )

bzopen() открывает bzip2 (.bz2) файл для чтения и записи.

Список параметров

filename

Имя открываемого файла.

mode

Аналогично функции fopen(), поддерживаются только 'r' (чтение) и 'w' (запись). Все остальное заставит bzopen вернуть FALSE.

Возвращаемые значения

Если открытие не удалось, bzopen() возвращает FALSE, иначе возвращается указатель на открытый файл.

Примеры

Пример #1 Пример использования bzopen()

<?php

$file = "/tmp/foo.bz2";
$bz = bzopen($file, "r") or die("Couldn't open $file for reading");

bzclose($bz);

?>

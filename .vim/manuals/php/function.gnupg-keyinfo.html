gnupg_keyinfo

(PECL gnupg >= 0.1)

gnupg_keyinfo — Returns an array with information about all keys that matches the given pattern

Описание

array gnupg_keyinfo ( resource $identifier , string $pattern )

Список параметров

identifier

Идентификатор gnupg, полученный из gnupg_init() или gnupg.

pattern

The pattern being checked against the keys.

Возвращаемые значения

Returns an array with information about all keys that matches the given pattern or FALSE, if an error has occurred.

Примеры

Пример #1 Procedural gnupg_keyinfo() example

<?php
$res = gnupg_init();
$info = gnupg_keyinfo($res, 'test');
print_r($info);
?>

Пример #2 OO gnupg_keyinfo() example

<?php
$gpg = new gnupg();
$info = $gpg -> keyinfo("test");
print_r($info);
?>

dcgettext

(PHP 4, PHP 5)

dcgettext — Запрашивает сообщение из другого каталога сообщений

Описание

string dcgettext ( string $domain , string $message , int $category )

Запрашивает перевод из другого каталога сообщений. Также позволяет указать категорию.

См. также gettext().

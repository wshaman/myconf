oci_lob_copy

(PHP 5, PECL OCI8 >= 1.1.0)

oci_lob_copy — Копирует содержание или часть содержания одного объекта LOB в другой

Описание

bool oci_lob_copy ([ int $length ] )

Копирует содержание или часть содержания объекта LOB в другой объект LOB. Параметр length указывает длину данных, которые следует скопировать.

Если вам необходимо скопировать определенную часть объекта, то для этого можно использовать oci_lob_seek(), чтобы передвинуть внутренние указатели объектов на нужные позиции.

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

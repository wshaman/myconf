sqlite_rewind

(PHP 5, PECL sqlite >= 1.0.0)

sqlite_rewind — Переходит к первой записи

Описание

bool sqlite_rewind ( resource $result )

sqlite_rewind() перемещает внутренний указатель результата запроса на первую запись. Возвращает FALSE если результат запроса не содержит записей, иначе TRUE.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Смотрите также

sqlite_next(), sqlite_current() и sqlite_seek().

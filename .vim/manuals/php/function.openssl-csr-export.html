openssl_csr_export

(PHP 4 >= 4.2.0, PHP 5)

openssl_csr_export — Exports a CSR as a string

Описание

bool openssl_csr_export ( resource $csr , string &$out [, bool $notext = true ] )

openssl_csr_export() takes the Certificate Signing Request represented by csr and stores it as ascii-armoured text into out, which is passed by reference.

Список параметров

csr

out

notext

Необязательный параметр notext влияет на детализацию сообщений вывода; если он установлен в FALSE, тогда в вывод добавляется дополнительная человекочитаемая информация. Значение по умолчанию notext является TRUE.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

  • openssl_csr_new() - Generates a CSR

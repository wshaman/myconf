gzclose

(PHP 4, PHP 5)

gzclose — Закрывает открытый gz-указатель

Описание

bool gzclose ( resource $zp )

Закрывает открытый gz-файл по его указателю.

Список параметров

zp

Указатель на gz-файл. Должен указывать на файл, успешно открытый gzopen().

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

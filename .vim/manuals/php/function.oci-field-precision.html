oci_field_precision

(PHP 5, PECL OCI8 >= 1.1.0)

oci_field_precision — Возвращает точность поля

Описание

int oci_field_precision ( resource $stmt , int $field )

Возвращает точность поля под номером field в результате запроса. Нумерация начинается с 1.

Для полей типа FLOAT точность больше нуля, а масштаб, получаемый с помощью oci_field_scale(), равен -127. Если точность поля равна нулю, то тип поля - NUMBER. Иначе же, тип поля может быть описан как NUMBER(precision, scale).

Замечание:

В версиях PHP ниже 5.0.0 эта функция называлась ocicolumnprecision(). В PHP 5.0.0 и выше ocicolumnprecision() является алиасом oci_field_scale(), поэтому вы можете продолжать использовать это имя, однако это не рекомендуется.

См. также oci_field_scale() и oci_field_type().

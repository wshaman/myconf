is_bool

(PHP 4, PHP 5)

is_bool — Проверяет, является ли переменная булевой

Описание

bool is_bool ( mixed $var )

Проверяет, является ли данная переменная булевой.

Список параметров

var

Проверяемая переменная.

Возвращаемые значения

Возвращает TRUE, если var является переменной типа boolean, FALSE в противном случае.

Примеры

Пример #1 Пример использования is_bool()

<?php
$a = false;
$b = 0;

// Если $a является булевой, условие выполняется
if (is_bool($a)) {
    echo "Yes, this is a boolean";
}

// Если $b не является булевой, условие не выполняется
if (is_bool($b)) {
    echo "Yes, this is a boolean";
}
?>

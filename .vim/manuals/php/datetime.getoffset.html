DateTime::getOffset

(PHP 5 >= 5.2.0)

DateTime::getOffset — Returns the timezone offset

Описание

Объектно-ориентированный стиль

public int DateTime::getOffset ( void )

Процедурный стиль

int date_offset_get ( DateTime $object )

Returns the timezone offset.

Список параметров

object

Только для процедурного стиля: Объект DateTime, возвращаемый date_create()

Возвращаемые значения

Returns the timezone offset in seconds from UTC on success или FALSE в случае возникновения ошибки.

Примеры

Пример #1 DateTime::getOffset() example

Объектно-ориентированный стиль

<?php
$winter = new DateTime('2010-12-21', new DateTimeZone('America/New_York'));
$summer = new DateTime('2008-06-21', new DateTimeZone('America/New_York'));

echo $winter->getOffset() . "\n";
echo $summer->getOffset() . "\n";
?>

Процедурный стиль

<?php
$winter = date_create('2010-12-21', timezone_open('America/New_York'));
$summer = date_create('2008-06-21', timezone_open('America/New_York'));

echo $winter->getOffset() . "\n";
echo $summer->getOffset() . "\n";
?>

Результат выполнения данных примеров:

-18000
-14400

Note: -18000 = -5 hours, -14400 = -4 hours.

NULL

Специальное значение NULL представляет собой переменную без значения. NULL - это единственно возможное значение типа NULL.

Замечание: Тип null был введен в PHP 4.

Переменная считается null, если:

  • ей была присвоена константа NULL.

  • ей еще не было присвоено никакого значения.

  • она была удалена с помощью unset().

Синтаксис

Существует только одно значение типа null - регистро-независимое ключевое слово NULL.

<?php
$var = NULL;       
?>

Смотрите также функции is_null() и unset().

Приведение к NULL

Приведение переменной к null удалит переменную и ее значение.

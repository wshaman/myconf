DateTime::format

(PHP 5 >= 5.2.0)

DateTime::format — Returns date formatted according to given format

Описание

Объектно-ориентированный стиль

public string DateTime::format ( string $format )

Процедурный стиль

string date_format ( DateTime $object , string $format )

Returns date formatted according to given format.

Список параметров

object

Только для процедурного стиля: Объект DateTime, возвращаемый date_create()

format

Format accepted by date().

Возвращаемые значения

Returns the formatted date string on success или FALSE в случае возникновения ошибки.

Примеры

Пример #1 DateTime::format() example

Объектно-ориентированный стиль

<?php
$date = new DateTime('2000-01-01');
echo $date->format('Y-m-d H:i:s');
?>

Процедурный стиль

<?php
$date = date_create('2000-01-01');
echo date_format($date, 'Y-m-d H:i:s');
?>

Результат выполнения данного примера:

2000-01-01 00:00:00

Примечания

This method does not use locales. All output is in English.

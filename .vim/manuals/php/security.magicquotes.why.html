Зачем использовать волшебные кавычки?

  • Полезно для начинающих Волшебные кавычки реализованы в PHP с целью осуществления помощи начинающим программистам в написании кода, который будет устойчив к опасностям. Хотя SQL инъекции до сих пор возможны при включенной директиве magic quotes, риск все же будет меньше.
  • Удобство При вставке данных в базу данных, волшебные кавычки преимущественно выполняют addslashes() для всех данных из Get, Post, и Cookie; и это происходит автоматически.

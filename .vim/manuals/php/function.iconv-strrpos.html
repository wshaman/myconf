iconv_strrpos

(PHP 5)

iconv_strrpos — Возвращает позицию последнего вхождения символа

Описание

string iconv_strrpos ( string $haystack , string $needle [, string $charset ] )

Возвращает позицию последнего вхождения needle в строку haystack.

Если подстрока needle не найдена, возвращает FALSE.

Внимание

Эта функция может возвращать как boolean FALSE, так и не-boolean значение, которое приводится к FALSE, например, 0 или "". За более подробной информации обратитесь к разделу Булев тип. Используйте оператор === для проверки значения, возвращаемого этой функцией.

Если haystack или needle не является строкой, он приводится к целому и трактуется как код символа.

В отличие от strpos(), iconv_strrpos() возвращает количество символов до подстроки, в не позицию вхождения в байтах. Символы считаются с учётом их кодировки charset.

См. также strrpos(), iconv_strpos() и mb_strrpos().
